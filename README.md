# WarframeDropTableHelper
一个将WARFRAME掉率数据转化为json的简单Java程序.程序生成的json文件主要为WARFRAME灰机wiki上的一些页面提供数据支持。

## 相关说明
* 该程序使用了“Genson”和“Apache POI”这两个库。
* 原数据需要先经过手动处理的方式整合成一个“droptable.xlsx”，然后交由本程序生成数个json文件。
